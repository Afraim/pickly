Getting Started

What is Pickly?

Pickly is a free, fast, privacy-first color picker tool. Upload images, drag & drop, or paste from clipboard to instantly extract color codes in HEX, RGB, HSL, CMYK, and LAB formats. All processing happens in your browser‚Äîno data is uploaded or tracked.

How do I use Pickly?

Step 1: Load an image by uploading, dragging, dropping, or pasting from clipboard.

Step 2: Hover your cursor (desktop) or touch (mobile) over the image to see the magnifier.

Step 3: Click or tap to pick the color at the center of the magnifier.

Step 4: Click the color in the palette to copy its code, or use the individual copy buttons for each format.

How does the magnifier work?

The circular magnifier shows a 100√ó100px zoomed view of the pixels under your cursor/finger. The center crosshair marks the exact pixel being sampled. This gives you pixel-perfect accuracy when picking colors. Magnification is 8√ó, so small details become clear.

Color Formats & Technology

What color formats does Pickly support?

HEX: #FF5733 (web standard, compact)

RGB: rgb(255, 87, 51) (Red, Green, Blue for screens)

HSL: hsl(13¬∞, 100%, 60%) (Hue, Saturation, Lightness for designers)

CMYK: cmyk(0%, 66%, 80%, 0%) (Cyan, Magenta, Yellow, Key for print)

HSV: hsv(13¬∞, 80%, 100%) (Hue, Saturation, Value)

LAB: lab(59, 43, 50) (Perceptually uniform color space)

What's the difference between RGB and HEX?

Both describe the same color, just in different formats:

RGB: Uses three numbers (0-255) for Red, Green, Blue intensity. Example: rgb(255, 87, 51)

HEX: Uses a 6-digit code (#RRGGBB). Same color: #FF5733

Use: HEX is more compact for web. RGB is what designers see when picking colors. Both work on screens.

When should I use HSL?

HSL is more intuitive for designers because it separates color, saturation, and brightness:

Hue (0-360¬∞): The color itself (red, blue, green, etc.)

Saturation (0-100%): How vivid the color is (0% = gray, 100% = pure)

Lightness (0-100%): How bright it is (0% = black, 50% = normal, 100% = white)

Use HSL when you want to quickly adjust brightness or desaturate a color.

When should I use CMYK?

Use CMYK for print projects only:

CMYK: Cyan, Magenta, Yellow, Key (Black)

Why: Printers use CMYK inks. RGB colors on screen won't match exactly when printed.

Workflow: Design in RGB/HEX for web. Convert to CMYK when sending to print.

Pickly automatically converts any color to CMYK for your convenience.

What's LAB color space?

LAB is a perceptually uniform color space where distance between colors matches human perception. It's useful for advanced color matching and accessibility. Most users don't need LAB; it's there for professionals.

Features & Exports

How do I export my color palette?

Click the "Export as JSON" button under "Picked Colors". This downloads a JSON file with all your colors and formats. You can:

  • Import into design tools
  • Share with team members
  • Use in web projects

Can I paste an image URL?

Not yet in the current version. Future updates will support URL input for faster color picking from web images.

Can I upload large images?

Yes, Pickly supports images of any size. However:

  • Large images (>10MB) may be slow on low-end devices
  • For best performance, use images under 5MB
  • Higher resolution = more accurate color picking

Can I pick multiple colors at once?

Yes! Click as many colors as you want. They'll appear in the "Picked Colors" palette. Each click adds a new color to your collection.

How do I clear my palette?

Click the "Clear" button at the top of the page. This removes all picked colors and clears the image. Your data is never saved server-side, so closing the browser also clears everything.

Troubleshooting

Clipboard paste isn't working. Why?

Clipboard paste requires HTTPS (secure connection). If you're on a local file (file://), paste may be blocked. Solutions:

  • Use the online version: https://afraim.github.io/pickly/
  • Use the Upload or Drag & Drop method instead
  • Run a local server with HTTPS

The magnifier isn't showing. Why?

The magnifier only appears when you load an image. Make sure to:

  1. Upload or paste an image first
  2. Move your cursor over the image (desktop)
  3. Or touch the image (mobile)

Copy to clipboard isn't working on mobile. Why?

Mobile browsers may have stricter clipboard permissions. Workarounds:

  • Ensure you're on HTTPS (not file://)
  • Grant clipboard permission when prompted
  • Use Safari or Chrome (most compatible)
